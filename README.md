# 305Vision

305Vision 是一个工业机器视觉软件平台，旨在实现 **工业相机图像采集、算法处理、视觉检测流程管理以及结果可视化**。该项目整合了算法调用、C#/C++ 混合编程及现场工程功能，适用于工业检测、质量控制和自动化产线场景。

## 功能特性

- **视觉检测流程管理**  
  支持多项目视觉检测流程搭建与管理，支持 **Opencv/深度学习Onnx模型调用**。
- **图像采集与处理**  
  支持工业相机实时采集图像，进行图像拼接、测量和算法处理。

## 技术栈

- **语言**: C#, C++  
- **界面框架**: WinForms  
- **工业相机 SDK**: 海康 / 通用工业相机  
- **算法集成**: Sherlock、OpenCV  
- **通信协议**: TCP/IP, Modbus, 串口, PLC(MC协议)  
- **调试工具**: Visual Studio, WinDbg, 系统日志
